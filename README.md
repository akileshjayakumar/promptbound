# Promptbound: The Five Trials of AI

## Overview

Promptbound is a retro pixel side-scroller about learning beginner AI engineering through play. You move through a series of interactive trials, meet in-world characters, collect rewards, and solve AI-themed challenges as part of the adventure.

## Gameplay

The game is built around five playable trials: Prompt Gate, Token Mine, Retrieval Library, Toolbelt Trial, and Agent Citadel. Each level teaches a core AI concept through movement, dialogue, puzzles, collectibles, badges, and challenge responses.

## Story And Theme

Promptbound follows an apprentice learning how to work with AI systems thoughtfully. The journey is about turning vague ideas into clear prompts, understanding tokens and retrieval, using tools well, and finally bringing those skills together in an agent-style challenge.
